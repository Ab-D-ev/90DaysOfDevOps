Q.) What is the Linux Command to ->
To view what's written in a file.





 #Solution 1:
 cat file_name
To change the access permissions of files.





 #Solution 2:
 chmod 777 file_name
To check which commands you have run till now.





 #Solution 3:
 history
To remove a directory/ Folder.





 #Solution 4:
 rm file_name
To create a fruits.txt file and to view the content.





 #Solution 5:
 nano fruits.txt #create file
 cat fruits.txt #view file
Add content in devops.txt (One in each line) - Apple, Mango, Banana, Cherry, Kiwi, Orange, Guava.




 #Solution 6:
 nano devops.txt # Add Apple, Mango, Banana, Cherry, Kiwi, Orange, Guava.
To Show only the top three fruits from the file.





 #Solution 7:
 head -3 devops.txt
To Show only the bottom three fruits from the file.





 #Solution 8:
 tail -3 devops.txt
